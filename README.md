# Vue-Gleo

A Gleo component for Vue.

## Demo

Available at [https://gleo.s3-website-zh.os.switch.ch](https://gleo.s3-website-zh.os.switch.ch)

## Development

```bash
npm install
npm run dev
```

Access at: `http://localhost:3001`

## Production

```bash
npm install
npm run build
```

Place the `dist` dir on your webserver of choice.
